// process · B2B 세계관 IP 제작

Process

메제웍스의 B2B 세계관 IP 제작 영역에서 작동하는 8단계 공정.
IP 데이터 수집에서 팬덤 액티비티 콘텐츠 제작까지 하나의 순서로 흐릅니다.

• 01 / 수집

IP 데이터 크롤링

IP Data Crawling

원천 IP, 공개 자료, 팬덤 반응, 기존 설정 자료를 관찰 가능한 데이터로 수집.

 • 02 / 구조화

키워드/세계관 라이브러링

Keyword · Universe Librarying

키워드, 개념, 관계, 사건, 장소를 추출해 세계관 지식 구조로 묶기.

 • 03 / 자료화

서연각 번역 및 자료집

Translation Dossier

외국어 원본과 참고 자료를 번역·각색·해설 가능한 자료집으로 정리.

 • 04 / 설정집

세계관 설정집/로어북

Universe Bible · Lorebook

라이브러링된 지식을 설정집, 로어북, 공개 위키, 책 구조로 편집.

 • 05 / 캐릭터

캐릭터 창작 및 시트

Character Creation · Sheet Guide

캐릭터를 서사, 욕망, 관계, 말투, 시각 요소, 운용 시트 단위로 제작.

 • 06 / 이야기

스토리텔링 100 생산

Storytelling 100

세계관과 캐릭터 시트를 바탕으로 참여형 장면과 시나리오 100개 생산.

 • 07 / IP 베이스

캐릭터 디자인 및 IP 베이스

Character Design · IP Base

캐릭터 디자인, 톤앤매너, 상품화 기준, 운영 규칙을 하나의 IP 베이스로 구축.

 • 08 / 실행

팬덤 액티비티 콘텐츠

Fandom Activity Contents

완성된 IP 베이스를 매일 참여 가능한 미션, 콘텐츠, 이벤트, 플랫폼 활동으로 전개.
