// process · B2B 세계관 IP 제작

Process

메제웍스의 B2B 세계관 IP 제작 영역에서 작동하는 다섯 프로세스.
크롤링·모니터링으로 들어온 자료가 세계관 IP로 정리되어 콘텐츠로 흐릅니다.

• 01 / 입력

서연각

Seoyeongak · 6 Steps

외국어 원본 → 번역·각색 자료 35종. Pride and Prejudice 사례.

 • 02 / 입력

캐릭터

Character · 12 Steps

시드 한 줄 → 한 인물 풀빌드 + 다섯 매체 출력.

 • 03 / 정리

라이브러링

Librarying · 4 Stages

흩어진 자료 → 옵시디안 Vault 1,500 표제어. 단일 진실 공급원.

 • 04 / 산출

로어북

LOREBOOK · 6 Phases

Vault → B5 양장 450p 백과사전 + 웹 공개본.

 • 05 / 산출

스토리100

Storytelling 100 · 7 Stages

Vault + 캐릭터 → 일상단면 시나리오 100편 별권.
