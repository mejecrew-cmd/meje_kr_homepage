MEJE Works의 B2B 세계관 IP 제작 영역에서 작동하는 다섯 프로세스.
세계관·캐릭터·100 시나리오는 그 자체가 목적이 아니라
100일 팬덤 액티비티 운영을 위한 입력 자산입니다.

// MEJE WORKS · END-TO-END PIPELINE

CRAWLING → MONITORING → ★ 세계관 IP 제작 (5 PROCESS) → 100일 팬덤 액티비티 운영 →

이 페이지는 가운데 단계 세계관 IP 제작의 5 프로세스를 다룹니다. 이 산출 자산이 다음 단계 100일 팬덤 액티비티 운영의 입력이 되며, 자체 IP 산출물(FEWK 시리즈 등)은 Works에서 확인할 수 있습니다.

다섯 프로세스 연재

각 프로세스는 14회 연재로 풀어 두었습니다. 자기완결적 1회 분량으로, 독자의 IP에 적용할 수 있도록 입력·산출·합격선·AI 작업 분담을 매 회 명시합니다.

01 / 입력
서연각 프로세스 연재 SEOYEONGAK · 書淵閣
외국어 원본을 번역·각색 가능한 자료로 변환하는 6단계. Pride and Prejudice를 사례로 14회.
14장 / 약 14만자
자세히 →
02 / 입력
캐릭터 프로세스 연재 CHARACTER BUILD
시드 한 줄에서 충분히 호흡하는 한 인물까지 12 단계. 다섯 매체(라이브러리·TRPG·시나리오·애니·웹소설) 출력까지.
14장 / 약 12만자
자세히 →
03 / 정리
라이브러링 연재 LIBRARYING
룰북·단편·기획서·외부 위키 같은 흩어진 자료에서 키워드를 끌어내 옵시디안 Vault 1,500 표제어 사전으로 묶는 4단계 파이프라인.
14장 / 약 17만자
자세히 →
04 / 산출
로어북 프로세스 연재 LOREBOOK
Vault 또는 IP 자료를 B5 양장 약 450p 출판본과 웹 공개본으로 묶는 6 Phase. 12부형 표준 위에 5종 변주.
14장 + 부록
자세히 →
05 / 산출
스토리100 프로세스 연재 STORYTELLING 100
Vault와 캐릭터 시트를 입력으로 100편의 일상단면 시나리오를 짓는 7단계. 세계관은 룰북으로 설명되지만 단편으로 살아납니다.
14장 / 약 17만자
자세히 →

다섯 프로세스가 공유하는 원칙

다섯 프로세스는 다른 산출물을 짜지만 같은 골격 위에 놓입니다. 제작자와 AI 작업 동료의 협업 4축, 단일 .md 소스, 매 단계의 합격선 게이트.

>_ COLLAB
제작자 + AI 작업 동료

매 단계에서 사람의 결정·압축·합격선 판단과 AI의 1차 산출·양식 변환·반복 작업을 분리해 둡니다. 협업 4축(규격·의도·기대 효과·한계)이 매 단계에서 같은 형식으로 짚입니다.

# SOURCE
단일 .md 소스

옵시디안 Vault 한 작업 폴더가 모든 산출의 단일 진실 공급원. 같은 자료가 출판 PDF로도, 정적 사이트로도, 다국어판으로도 변환됩니다. 다른 매체로 옮길 때 자료가 두 갈래로 갈리지 않습니다.

// GATE
합격선 게이트

매 단계의 진입에 자동 검사와 사람 검수가 함께 작동합니다. 한 단계의 결함이 다음 단계 전체로 번지지 않도록, 임시 우회 없이 매 게이트를 의무로 둡니다.
