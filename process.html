Process

세계관 IP 제작 파이프라인 · 8단계 제작 공정

MEJE Works의 B2B 세계관 IP 제작은 데이터 수집에서 팬덤 액티비티 콘텐츠 제작까지 이어집니다.
세계관·캐릭터·100 시나리오는 각각 따로 흩어진 결과물이 아니라
팬덤이 매일 참여할 수 있는 IP 운영 자산으로 연결됩니다.

// MEJE WORKS · END-TO-END PIPELINE

IP DATA CRAWLING → LIBRARYING → SEOYEONGAK → LOREBOOK → CHARACTER SHEETS → STORY 100 → IP BASE → FANDOM ACTIVITY →

이 페이지는 IP 데이터 크롤링 기술 → 키워드/세계관 라이브러링 → 서연각 번역 및 자료집 → 세계관 설정집/로어북 제작 → 캐릭터 창작 및 시트 제작 디자인 가이드 → 스토리텔링 100 생산 → 캐릭터 디자인 및 IP 베이스 구축 → 팬덤 액티비티 콘텐츠 제작 순서를 기준으로 정리합니다. 자체 IP 산출물(FEWK 시리즈 등)은 Works에서 확인할 수 있습니다.

8단계 공정이 공유하는 원칙

각 단계는 다른 산출물을 짜지만 같은 골격 위에 놓입니다. 제작자와 AI 작업 동료의 협업 4축, 단일 .md 소스, 매 단계의 합격선 게이트.

>_ COLLAB
제작자 + AI 작업 동료

매 단계에서 사람의 결정·압축·합격선 판단과 AI의 1차 산출·양식 변환·반복 작업을 분리해 둡니다. 협업 4축(규격·의도·기대 효과·한계)이 매 단계에서 같은 형식으로 짚입니다.

# SOURCE
단일 .md 소스

옵시디안 Vault 한 작업 폴더가 모든 산출의 단일 진실 공급원. 같은 자료가 출판 PDF로도, 정적 사이트로도, 다국어판으로도 변환됩니다. 다른 매체로 옮길 때 자료가 두 갈래로 갈리지 않습니다.

// GATE
합격선 게이트

매 단계의 진입에 자동 검사와 사람 검수가 함께 작동합니다. 한 단계의 결함이 다음 단계 전체로 번지지 않도록, 임시 우회 없이 매 게이트를 의무로 둡니다.
